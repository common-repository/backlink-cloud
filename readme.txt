The Backlink Cloud plugin uses the backlink cloud server for listing, creating, and validating link reciprocation.

== Description ==

This plugin allows your site to be a backlink cloud reciprocation host. Backlink Cloud Reciprocation hosts become targets for massive backlinks from prevailing websites. When a specific parameter is passed to your site it will generate information to send a backlink back to the site that has linked to you.


== Installation ==

1. Install plugin to the wp-content/plugin directory 
2. Activate the plugin. When the plugin activates it registers your site to be listed in the Backlink Cloud.

== Frequently Asked Questions ==

= Why should I use this plugin? =

Backlinks are the bread and butter to SEO and driving traffic to your website. If you wish to effortlessly promote your website then this is a very handy tool.

= I can't get the plugin to work! =

I am more than willing to help you. Please visit the [Backlink Cloud](http://www.backlinkcloud.com) and post a request for help.

== Screenshots ==

1. The backlink is the title, screenshot, and description of the website.
2. The administration section under Settings -> Backlink Cloud will display the last 10 reciprocation requests. You may also specify domains you wish not to reciprocate to.
